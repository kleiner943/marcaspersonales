Bug crítico — cálculo de hora

El problema que ves (-2:00 — Fuera de horario) es porque cuando UTC es la 1AM o 2AM, restar 5 da número negativo. La función nunca entra al rango 6–22.
Código actual (roto):
function isTradingHours() { const now = new Date(); const hour = now.getUTCHours() - 5; // ← PROBLEMA: puede dar -4, -3, -2... return hour >= CONFIG.tradingStart && hour < CONFIG.tradingEnd; }
Corrección (copiar y reemplazar):
function isTradingHours() { const now = new Date(); // Hora Ecuador correcta (UTC-5), maneja negativo let hour = now.getUTCHours() - 5; if (hour < 0) hour += 24; return hour >= CONFIG.tradingStart && hour < CONFIG.tradingEnd; }

Cómo funciona EVA + TESLA

TESLA escribe señales y trades a un archivo tesla_pipe.json. EVA lo monitorea cada 30 segundos y cuando hay una operación nueva, te manda un WhatsApp automático como dueño.
Agrega esto al final de executeTrade() en tesla.js:
// ── Notificar a EVA via pipe ────────────────── const pipePath = require('path').join('C:/bot/temp', 'tesla_pipe.json'); const pipes = require('fs').existsSync(pipePath) ? JSON.parse(require('fs').readFileSync(pipePath)) : []; pipes.push({ action, symbol, pnl: pnl ? pnl.toFixed(2) : null, reason, price: order.price, timestamp: new Date().toISOString(), read: false }); if (pipes.length > 50) pipes.shift(); require('fs').writeFileSync(pipePath, JSON.stringify(pipes, null, 2)); // ─────────────────────────────────────────────

Agrega esto en bot-whatsapp.js (monitor de TESLA)

// ── Monitor TESLA → EVA WhatsApp ───────────── const TESLA_PIPE = 'C:/bot/temp/tesla_pipe.json'; setInterval(() => { try { if (!fs.existsSync(TESLA_PIPE)) return; const signals = JSON.parse(fs.readFileSync(TESLA_PIPE)); let updated = false; for (const sig of signals) { if (sig.read) continue; sig.read = true; updated = true; const emoji = sig.action === 'BUY' ? '🟢' : '🔴'; const pnlTxt = sig.pnl ? (parseFloat(sig.pnl) >= 0 ? `✅ +$${sig.pnl}` : `❌ $${sig.pnl}`) : ''; const msg = `${emoji} *TESLA* — ${sig.action} ${sig.symbol}\n` + `💲 Precio: $${sig.price?.toFixed(2)}\n` + `📌 Razón: ${sig.reason}\n` + `${pnlTxt ? pnlTxt + '\n' : ''}` + `⏰ ${new Date(sig.timestamp).toLocaleTimeString('es-EC')}`; client.sendMessage(process.env.OWNER_NUMBER, msg); } if (updated) fs.writeFileSync(TESLA_PIPE, JSON.stringify(signals, null, 2)); } catch(e) {} }, 30000); // cada 30 segundos // ─────────────────────────────────────────────
